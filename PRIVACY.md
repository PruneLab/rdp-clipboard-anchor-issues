# Privacy Policy / 隐私政策

**Last Updated / 最后更新:** March 11, 2026

## English

### 1. Data Collection & Processing
**RDP Clipboard Anchor** ("the Application") operates primarily as a local clipboard history manager.
- **Clipboard Access:** The Application monitors and reads the system clipboard to cache texts, images, and files copied during an active RDP session or on your local machine.
- **Local Cache:** All intercepted clipboard data is temporarily stored in your local application data folder (`%LocalAppData%\RdpClipboardAnchor\Cache`). The Application never uploads or transmits this data to any remote server or third party. You have full control to clear this cache through the Application's settings.
- **Usage Statistics:** The Application maintains basic usage counters (e.g., number of successful restorations) solely stored locally on your device to trigger milestone prompts. No usage telemetry is sent over the network.

### 2. Internet Access
The Application does not transmit personal data over the internet. Internet access is only utilized for:
- Querying your product license status via the official Microsoft Store API.
- Opening the developer's Microsoft Store page when you click the "Support Developer" or "Purchase" buttons.
- Opening the project's GitHub repository page when you click the "Report an Issue" link in the "About" dialog.

### 3. Third-Party Services
The Application does not use any third-party analytics, advertising services, or tracking SDKs. It relies solely on the Microsoft Store for secure licensing.

### 4. Contact Us
If you have any questions about this Privacy Policy or data security concerns, please contact us via the project's GitHub issue tracker:
https://github.com/LuShuchen/rdp-clipboard-anchor-issues

## 中文 (Chinese)

### 1. 数据收集与处理
**RDP Clipboard Anchor**（以下简称“本应用”）主要作为本地剪贴板历史管理器运行。
- **剪贴板访问：** 本应用会监听并读取系统剪贴板，以缓存您在远程桌面（RDP）会话期间或本地计算机上复制的文本、图片和文件。
- **本地缓存：** 所有拦截的剪贴板数据均临时存储在您的本地应用数据文件夹中（`%LocalAppData%\RdpClipboardAnchor\Cache`）。本应用绝不会将这些数据上传或传输至任何远程服务器或第三方。您可以通过应用设置随时完全清理这些缓存。
- **使用统计：** 本应用仅在您的设备本地维护基本的使用次数计数器（如成功恢复文集的次数），用于触发特定的里程碑提示。不会通过网络发送任何遥测数据。

### 2. 互联网访问
本应用不会通过互联网传输您的任何个人数据。互联网访问仅用于以下场景：
- 通过微软官方的 Microsoft Store API 查询您的软件许可状态（试用版/完整版）。
- 当您点击“支持开发者”或“购买”按钮时，打开您的默认浏览器跳转至开发者的 Microsoft Store 页面。
- 当您点击关于页面的“报告问题”链接时，打开本项目的 GitHub 仓库页面。

### 3. 第三方服务
本应用不使用任何第三方数据分析、广告分发或用户跟踪 SDK。软件的授权验证完全依赖微软官方安全的 Microsoft Store 机制。

### 4. 联系我们
如果您对本隐私政策或数据安全有任何疑问，请通过项目的 GitHub issues 页面联系我们：
https://github.com/LuShuchen/rdp-clipboard-anchor-issues
