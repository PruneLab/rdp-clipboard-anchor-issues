### 🛑 당신에게 필요한 RDP «앵커(닻)»
원격 서버에서 중요한 코드 스니펫, 이미지 또는 파일을 복사했는데 갑자기 RDP 세션이 멈추거나 끊긴 적이 있으신가요? 연결을 다시 시작하고 로컬에 붙여넣으려 하지만... 아무것도 없습니다. 클립보드가 비어 있는 것입니다.

**RDP Clipboard Anchor**는 바로 이러한 고충을 해결하기 위해 탄생했습니다. 로컬 PC에서 안전하게 실행되며 세션 간의 복사 이벤트를 즉각적이고 자동으로 캐시합니다. 원격 데스크톱(RDP) 연결이 멈추거나 충돌하더라도 복사된 데이터는 함께 사라지지 않습니다. 로컬에 안전하게 «고정(앵커)»되어 어디든 붙여넣거나 드래그할 수 있도록 준비됩니다.

### 🚀 강력한 Win+V 대체재
원격 세션 보호를 넘어, Windows 기본 클립보드 기록(Win+V)을 완벽하게 대체하는 강력한 도구로 진화했습니다. 개발자와 파워 유저를 위해 설계되었으며 다음을 제공합니다:
* **풍부한 포맷 지원:** 표준 도구보다 복잡한 데이터 형식, 이미지 및 파일을 훨씬 더 잘 처리합니다.
* **매끄러운 드래그 앤 드롭:** 기록에서 다른 응용 프로그램이나 폴더로 항목을 직접 드래그할 수 있습니다.
* **스마트 캐시 관리:** 사용자 지정 가능한 할당량 및 자동화된 지능형 정리 로직으로 스토리지를 최적화된 상태로 유지합니다.
* **빠른 작업:** 항목을 두 번 클릭하여 즉시 열거나 실행할 수 있습니다.
* **개인정보 우선:** 원클릭으로 모니터링을 일시 중지하여 민감한 작업을 보호합니다.
* **분리된 아이콘 체계:** Windows 네이티브 파일 아이콘에 소스 배지(로컬 vs 원격)를 결합하여 복사한 콘텐츠의 출처를 즉시 식별할 수 있습니다.
* **네이티브 환경:** Windows 11과 완벽하게 일치하는 깔끔하고 눈에 띄지 않는 WinUI 3 인터페이스를 제공합니다.

### 🛡️ 100% 오프라인 및 안전성
클립보드에는 매우 민감한 정보가 포함되는 경우가 많습니다. 이 응용 프로그램은 인터넷 액세스 및 원격 측정 없이 100% 로컬로 작동합니다. 또한 보안 마커를 사용하는 주요 암호 관리자(예: 1Password)의 임시 클립보드 이벤트를 자동으로 무시하여 개인 정보를 지능적으로 보호합니다. 복사한 내용은 철저히 사용자의 통제하에 유지됩니다.

자세한 내용은 [개인정보 처리방침](https://prunelab.net/ko/products/rdp-clipboard-anchor/privacy)을 확인해 주세요.

## 📦 사용 방법

1. **RDP Clipboard Anchor**를 실행하면 백그라운드에서 클립보드를 자동으로 모니터링합니다.
2. 로컬 또는 원격 데스크톱에서 텍스트, 이미지 또는 파일을 `Ctrl+C`로 복사하면 기록 목록에 즉시 나타납니다.
3. 다른 창으로 드래그, 더블 클릭으로 열기, 또는 현재 클립보드로 복원할 수 있습니다!
4. 원격 데스크톱(RDP) 사용 시 연결이 예기치 않게 끊어져도 복사한 데이터가 로컬에 안전하게 고정되어 유지됩니다.
