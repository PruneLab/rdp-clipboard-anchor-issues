**強力な Win+V 代替ツール。豊富なフォーマット（ファイル、画像、テキスト）に対応し、RDP 切断時のデータ消失を自動的に防ぎます。100% オフラインで安全。**

### 🛑 あなたが必要とする RDP の「アンカー（錨）」
リモートサーバー上で重要なコードの断片、画像、またはファイルをコピーした直後に、RDPセッションが突然フリーズしたり切断されたりした経験はありませんか？再接続してローカルで貼り付けようとすると…何もありません。クリップボードは空です。

**RDP Clipboard Anchor** は、まさにこのペインポイントを解決するために誕生しました。ローカルマシン上で安全に動作し、セッション間のコピーイベントを即座に自動キャッシュします。リモートデスクトップ（RDP）接続がハングアップしたりクラッシュしたりしても、コピーしたデータが一緒に消えることはありません。ローカルに安全に「アンカー（固定）」され、いつでもどこにでも貼り付けたりドラッグしたりできる状態を保ちます。

### 🚀 優れた Win+V 代替ツール
リモートセッションを救うだけでなく、Windows標準のクリップボード履歴（Win+V）を置き換える強力なツールへと進化しました。開発者やパワーユーザー向けに設計されており、以下の機能を提供します：
* **リッチフォーマットのサポート:** 複雑なデータ型、画像、ファイルを標準ツールよりもはるかに適切に処理します。
* **シームレスなドラッグ＆ドロップ:** 履歴から他のアプリケーションやフォルダーへ、アイテムを直接ドラッグできます。
* **スマートなキャッシュ管理:** カスタマイズ可能なクォータと自動化されたインテリジェントなクリーンアップロジックにより、ストレージを最適に保ちます。
* **クイックアクション:** アイテムをダブルクリックするだけで、瞬時に開いたり実行したりできます。
* **プライバシー優先:** ワンクリックでモニタリングを一時停止し、機密性の高い操作を保護します。
* **独立したアイコンシステム:** Windowsネイティブのファイルアイコンにソースバッジ（ローカル vs リモート）を組み合わせ、コピー元を一目で識別できます。
* **ネイティブな体験:** Windows 11と完璧に調和する、洗練された控えめな WinUI 3 インターフェース。

### 🛡️ 100% オフライン & セキュア
クリップボードには、多くの場合、機密性の高い情報が含まれています。このアプリケーションは完全にローカルで動作し、インターネットアクセスはなく、テレメトリーも一切ありません。さらに、主要なパスワードマネージャー（1Passwordなど）が使用する安全なマーカーが付与された一時的なクリップボードイベントを自動的に無視することで、プライバシーをインテリジェントに保護します。コピーした内容は、厳格にあなたの管理下に置かれます。

詳細については、[プライバシーポリシー](https://prunelab.net/ja/products/rdp-clipboard-anchor/privacy)をご確認ください。

## 📦 使い方

1. **RDP Clipboard Anchor** を起動すると、バックグラウンドでクリップボードを自動監視します。
2. ローカルでもリモートデスクトップでも、テキスト、画像、ファイルを `Ctrl+C` でコピーするだけで、履歴リストに自動的に表示されます。
3. 他のウィンドウにドラッグ、ダブルクリックで開く、または現在のクリップボードに復元できます！
4. リモートデスクトップ（RDP）使用中は、接続が予期せず切断されても、コピーしたデータがローカルに安全にアンカー（固定）されたまま残ります。
