**Мощная альтернатива Win+V с поддержкой сложных форматов (файлы, изображения, текст) и автоматическим кэшированием для предотвращения потери данных при обрывах RDP. 100% оффлайн и безопасно.**

### 🛑 RDP «ЯКОРЬ», КОТОРЫЙ ВАМ НУЖЕН
Случалось ли вам скопировать важный фрагмент кода, изображение или файл на удаленном сервере, после чего сессия RDP внезапно зависала или обрывалась? Вы перезапускаете соединение, пытаетесь вставить скопированное на локальном ПК, и... ничего. Буфер обмена пуст.

**RDP Clipboard Anchor** был создан именно для решения этой проблемы. Безопасно работая на вашем локальном компьютере, он мгновенно и автоматически кэширует события копирования между сессиями. Когда подключение к удаленному рабочему столу (RDP) зависает или дает сбой, скопированные данные не исчезают вместе с ним. Они надежно «заякорены» локально и готовы к вставке или перетаскиванию куда угодно.

### 🚀 Превосходная альтернатива Win+V
Помимо спасения удаленных сессий, программа превратилась в мощную замену встроенного журнала буфера обмена Windows (Win+V). Созданная для разработчиков и опытных пользователей, она предлагает:
* **Поддержку сложных форматов:** Обрабатывает сложные типы данных, изображения и файлы гораздо лучше, чем стандартные инструменты.
* **Бесшовный Drag & Drop:** Перетаскивайте элементы прямо из истории в другие приложения или папки.
* **Умное управление кэшем:** Оптимизируйте хранилище с помощью настраиваемых квот и автоматической интеллектуальной логики очистки.
* **Быстрые действия:** Дважды щелкните по элементу, чтобы мгновенно открыть или выполнить его.
* **Конфиденциальность прежде всего:** Приостановите отслеживание одним кликом для защиты конфиденциальных операций.
* **Декаплинговая иконография:** Нативные файловые значки Windows с маркерами источника (Локальный vs Удаленный) для мгновенного определения происхождения скопированного содержимого.
* **Нативный интерфейс:** Чистый, ненавязчивый интерфейс WinUI 3, который идеально сочетается с Windows 11.

### 🛡️ 100% Оффлайн и Безопасно
Ваш буфер обмена часто содержит строго конфиденциальную информацию. Это приложение работает полностью локально, без доступа к Интернету и без телеметрии. Более того, оно интеллектуально заботится о вашей конфиденциальности, автоматически игнорируя временные события буфера обмена от основных менеджеров паролей (таких как 1Password), использующих маркеры безопасности. ## 📦 Как использовать

1. Запустите **RDP Clipboard Anchor** — он автоматически отслеживает буфер обмена в фоновом режиме.
2. Будь то на локальном компьютере или на удаленном рабочем столе, нажмите `Ctrl+C` для любого текста, изображения или файла. Содержимое мгновенно появится в списке истории.
3. Перетаскивайте в другие окна, дважды щелкните для открытия или восстановите в текущем буфере обмена в любое время!
4. При использовании удаленного рабочего стола (RDP), даже если соединение неожиданно прервется, скопированные данные останутся надежно «заякорены» локально.
