**強大的 Win+V 高階平替。完美支援檔案、圖片與文字等豐富格式，自動快取並防止 RDP 遠端桌面斷線導致的資料遺失。100% 離線，安全可靠。**

### 🛑 你需要的 RDP 守護錨點
您是否曾在遠端伺服器複製了一段關鍵的程式碼、圖片或檔案，結果 RDP 連線突然卡死或斷線？當您重新連線準備在本地端貼上時，結果……什麼都沒有。剪貼簿是空的。

**RDP Clipboard Anchor** 正是為了解決這個痛點而生。它安全地運行在您的本地端電腦上，自動且即時地快取您跨工作階段的複製事件。當您的遠端桌面 (RDP) 連線卡住或崩潰時，您複製的資料不會隨之消失。它會被安全地「錨定」在本地，隨時準備貼上或拖曳到任何地方。

### 🚀 原生 Win+V 的完美高階平替
不僅僅是 RDP 伴侶，它更是一款專為開發者和效率控打造的高階剪貼簿管理中樞，全面超越 Windows 自帶的 Win+V：
* **豐富格式支援：** 比標準工具更出色地處理複雜的資料類型、圖片和檔案。
* **無縫拖放：** 直接從歷史記錄中將項目拖曳到其他應用程式或資料夾中。
* **智慧快取管理：** 透過可自訂的配額以及自動、智慧的清理邏輯，讓您的儲存空間保持最佳化。
* **快速操作：** 雙擊項目即可立即開啟或執行它們。
* **隱私優先：** 一鍵暫停監控，保護敏感操作場景。
* **解耦圖示體系：** 提取 Windows 原生檔案圖示，搭配右下角來源角標（本機 vs 遠端），一眼識別內容來源。
* **原生級 UI 體驗：** 基於最新 Windows App SDK (WinUI 3) 打造，完美融入 Windows 11 現代設計語言，輕量且優雅。

### 🛡️ 100% 純本地與絕對安全
您的剪貼簿經常包含高度機密的資訊。本應用程式完全在本地端運行，無網際網路存取權限，且零遙測。此外，它能智慧地尊重您的隱私，自動忽略來自主要密碼管理員（如 1Password）使用安全標記的暫時性剪貼簿事件。您複製的內容完全掌握在您自己手中。

## 📦 使用指南

1. 開啟 **RDP Clipboard Anchor**，它會自動在背景監控剪貼簿。
2. 無論是在本機還是遠端桌面中，按下 `Ctrl+C` 複製任何文字、圖片或檔案，內容都會自動出現在歷史記錄中。
3. 隨時拖曳到其他視窗、雙擊快速開啟，或恢復到目前剪貼簿！
4. 搭配遠端桌面 (RDP) 使用時，即使連線意外中斷，已複製的資料仍安全錨定在本地。
